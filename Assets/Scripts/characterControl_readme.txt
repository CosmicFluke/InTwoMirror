Use scene "controller_scene" to see script set up in action

Movement:
Move player with WASD (assumes camera faces +Z direction)


Camelot sound:
Currently has 5 slots
Use up/down arrow to select the tone
Press "E" to play the tone (will play while pressed)
Camelot souds are currently set to the same tone, please change in public variables

Secondary sound
Press "F" to play secondary sound
Can only interact if an object with tag "Interactive" is within range
Range can be set with "Max Action Distance" variable
